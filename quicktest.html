Mat-O-Wahl - Konfigurationstest / Configuration test

DE: Dieser Test liest alle wichtigen Einstellungen aus und liefert so einen schnellen Überblick, ob man vielleicht einen Fehler in der Konfiguration gemacht hat.

EN: This test reads all important settings and gives you a quick overview if there's a mistake in the configuration.

Fehlerbericht / Error Report

DE: Wenn diese Meldung auch nach mehreren Minuten nicht verschwindet und JavaScript aktiviert ist, haben Sie entweder einen sehr langsamen Rechner, einen sehr umfangreichen Fragebogen oder ein echtes Problem in den Definitions-Dateien. z.B. eine Einstellung, die nicht sauber mit "Anführungszeichen" eingeschlossen ist oder eine fehlende Einstellung. Zum Beispiel:

Falsch: Fehlendes Anführungszeichen am Anfang oder Ende.
var fileQuestions = Fragen.csv";
var fileQuestions = "Fragen.csv;

Falsch: Eine vollkommen fehlende Einstellung.
var fileQuestions =

Falsch: Eine Datei ohne Dateiendung (obwohl diese vorhanden ist) oder Groß- und Kleinschreibung. Dateiendungen werden im Windows-Explorer normalerweise ausgeblendet. :(
var fileQuestions = "Fragen";

Richtig:
var fileQuestions = "Fragen.csv";

EN: If this message is still showing after some minutes and JavaScript is active, you have either a very slow computer, a very complex questionnaire or a real problem with the preferences like a setting, that is not enclosed by "quotes" or one of the following wrong settings:

Wrong: Missing quote at the beginning or end.
var fileQuestions = Fragen.csv";
var fileQuestions = "Fragen.csv;

Wrong: A missing setting
var fileQuestions =

Wrong: A file without file extension (even thoug it exists) or a confusion with capital letters. Windows-Explorer usually hides the file extensions. :(
var fileQuestions = "Fragen";

Right:
var fileQuestions = "Fragen.csv";
